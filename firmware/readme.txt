Firmware archive:

The firmware for kantor and millitron is for Michrochips old MPLab IDE Software
and will probably not changed anymore.

For rhea  MPLAB-X suite is used as software development kit and still in development, a more actual one could be under the escher development folders. (email author for more actual information)

see http://microchip.com/mplab, http://microchip.com/mplabx

Note: Since project files in MPLABX are path senesitive, there maybe some path magic is needed to compile it.
